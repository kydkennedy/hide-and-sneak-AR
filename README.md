# hide-and-sneak-AR
 A simple AR game made with Blender, Three.js, and jsartoolkit5. The game uses image recognition to place 3D objects in the real world. The player has to find the hidden objects and sneak past virtual guards to win the game. 
 ## How to play
 -1. Download Hide&Sneak AR from app store or play store.
    -2. Open the app and point the camera at a marker. This will be the gate player have to sneak past to win the game. 
    -3. Once the gate is placed, the player can move around the real world and look for hidden objects specified on their screen. 
    -4. Once the player finds the hidden objects, they have to sneak past the virtual guards at the gate to win the game. 
## Game Area
The game area can be defined by players by placing the gate at the entrance of the game area. The game area can be any size as long as the gate is placed at the entrance. Hidden objects are placed randomly by distance within the game area. 
*Note: The larger the area, the more hidden objects there will be and the greater distance between the hidden objects.
## Game Objects
This game has 3 types of game objects:
-1. Hidden objects: These are the objects the player has to find. They are placed randomly within the game area. 
    -2. Virtual guards: These are the virtual guards that the player has to sneak past to win the game. They are placed at the gate. 
    -3. Gate: This is the gate that the player has to sneak past to win the game. It is placed at the entrance of the game area.
## Game Modes
This game has 2 game modes:
-1. Single player: The player has to find the hidden objects and sneak past the virtual guards to win the game. 
    -2. Multiplayer: The player has to find the hidden objects and sneak past the virtual guards to win the game. The player can play with other players in the same game area. The first to sneak past the virtual guards wins the game.
## Game Types
Hide&Sneak AR has 3 game types:
-1. Time Trial: Here the player has to find all hidden objects and sneak past the virtual guards before the time runs out.
-2. Survival: Here the player has to find all hidden objects within the game area, with increasing waves of guards patroling the game area. They do not have to bring object back to the entrance gate to win the game. The player has to survive as long as possible without getting caught by guards within the game area. The game ends once the player gets caught. Here higher score means longer survival time.
-3. Regular: Here the player has to find all hidden objects and sneak past the virtual guards to win the game. The player has to bring the hidden objects back to the entrance gate to win the game. The player has to bring all hidden objects back to the entrance gate to win the game. There is no score in this game mode.

## Game Settings
This game has 3 game settings:
-1. Game Area: The player can define the game area by placing the gate at the entrance of the game area. The game area can be any size as long as the gate is placed at the entrance. Hidden objects are placed randomly by distance within the game area. 
    -2. Game Mode: The player can choose between single player, multiplayer, time trial, survival, and regular game modes. 
    -3. Game Type: The player can choose between time trial, survival, and regular game types.
